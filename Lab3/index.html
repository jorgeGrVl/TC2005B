Preguntas HTML

¿Cuál es la diferencia entre Internet y la World Wide Web?

Internet es la infraestructura global de redes, mientras que la World Wide Web es un sistema de información basado en hipertexto que opera sobre Internet.

¿Cuáles son las partes de un URL?

Un URL se compone de protocolo, dominio, ruta, parámetros y anclajes.

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

GET obtiene datos, HEAD recupera encabezados, POST envía datos, PUT actualiza recursos, PATCH modifica parcialmente y DELETE elimina recursos.

¿Qué método HTTP se debe utilizar al enviar un formulario HTML?

Se usa POST, porque envía datos de manera segura y no se muestran en la URL.

¿Qué método HTTP se utiliza para acceder a una página web?

Se usa GET para recuperar el contenido de una página.

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto?

Significa que la solicitud fue exitosa y no hubo errores.

¿Es responsabilidad del desarrollador corregir un error 404?

No necesariamente, ya que un error 404 indica que el recurso solicitado no existe, pero puede ser responsabilidad del administrador del contenido.

¿Es responsabilidad del desarrollador corregir un error 500?

Sí, ya que un error 500 indica un problema en el servidor que debe ser corregido por el desarrollador.

¿Qué significa que un atributo HTML5 esté depreciado?

Significa que ya no es recomendable su uso y puede ser eliminado en futuras versiones. Ejemplo: <font> y <center> de HTML4.

¿Cuáles son las diferencias entre HTML4 y HTML5?

HTML5 introduce etiquetas semánticas, soporte multimedia, almacenamiento local y una mejor gestión de formularios.

¿Qué componentes tiene una tabla?

Una tabla incluye <table>, <tr>, <th>, <td> y opcionalmente <caption>, <thead>, <tbody>, <tfoot>.

¿Cuáles son los principales controles de una forma HTML5?

Incluyen input (text, email, password, number, date), textarea, select, checkbox, radio y button.

¿Qué tanto soporte HTML5 tiene el navegador que utilizas?

Google Chrome tiene un alto soporte para HTML5, implementando la mayoría de sus características y actualizándose regularmente.

¿Cuál es el ciclo de vida de los sistemas de información?

Incluye planificación, análisis, diseño, implementación, pruebas, mantenimiento y retirada.

¿Cuál es el ciclo de desarrollo de sistemas de información?

Consiste en análisis de requerimientos, diseño del sistema, implementación, pruebas, despliegue y mantenimiento.

Preguntas CSS

Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

Usa !important solo como último recurso para evitar problemas de mantenimiento.

Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

Elige cuidadosamente la imagen de fondo para optimizar rendimiento, legibilidad y adaptabilidad.

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

Prefiere % para diseños responsivos, px para precisión y evita pt en web.

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

La versión minimizada de CSS mejora el rendimiento al reducir el tamaño del archivo y acelerar la carga.
